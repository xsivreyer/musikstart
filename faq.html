Installatie

Hoe kan ik de software installeren?
  • Download de laatste versie v/d software
  • Pak het zip-bestand uit, bijvoorbeeld in de 'Opstarten' map. (Windowstoets+R: shell:startup)
  • Start de software
Hoe kan ik de software verbergen?
  • Klik op Window
  • Klik op Tray (startup)
  • De software wordt nu bij het opstarten automatisch verborgen
De mixer interface wordt niet herkend. Wat nu?
  • Controleer de bekabeling
  • Controleer of de drivers automatisch zijn geinstalleerd. Dit kan in Windows bij 'Apparaatbeheer'. Bij 'Poorten' moet daar een 'USB Serial Port' staan of een 'USB SERIAL CH340'. Als deze hier niet terug te vinden is, dan kan deze waarschijnlijk worden gevonden onder 'Andere apparaten' en moeten de drivers handmatig worden toegevoegd, deze zijn hier te downloaden. Vooral bij Windows 7 lijkt het altijd nodig om deze te installeren.
