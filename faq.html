Vaak gestelde vragen

Hoe kan ik de software installeren? Hoe kan ik de software verbergen?

Installatie

Vraag Hoe kan ik de software installeren?
Antwoord
  • Download de laatste versie v/d software
  • Pak het zip-bestand uit, bijvoorbeeld in de 'Opstarten' map. (Windowstoets+R: shell:startup)
  • Start de software
Vraag Hoe kan ik de software verbergen?
Antwoord
  • Klik op Window
  • Klik op Tray (startup)
  • De software wordt nu bij het opstarten automatisch verborgen
